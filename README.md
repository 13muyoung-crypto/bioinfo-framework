# 生信分析导航台 (Bioinfo Navigator)

基于杨英淇《零代码从框架入门生信》(2026.3.31) 的交互式生信分析导航系统。

不是问答机器人——是一张按"找差异→解释差异→诊与治"框架组织的地图，
带状态记忆、检查清单、标准图参考和排错引擎。

## 三层状态机

- **导航模式**：带你走完整分析流程，每步有检查清单+图参考+路由规则
- **排错模式**：四层优先级+六组学替代矩阵+通用替代链，系统排查问题
- **查阅模式**：84张图+25种分析方法的速查参考

## 安装

将本仓库克隆到 `$CODEX_HOME/skills/bioinfo-navigator/` 目录下。
